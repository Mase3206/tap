# Mase3206 Tap

## Formulae

- [`past`](https://github.com/hypercomrade/past) - HEAD-only
- [`seconv`](https://github.com/SubtitleEdit/subtitleedit/tree/main/src/seconv) (part of [SubtitleEdit](https://www.nikse.dk/subtitleedit)) - A modern, headless command-line utility for batch converting subtitle files between formats.

## Casks

- [Detexify](https://detexify.kirelabs.org/#/mac): `detexify-next` ([kirel/detexify-next](https://github.com/kirel/detexify-next)) - A menu bar app to help you find the LaTeX command for a symbol you can draw but not name.
- [SubtitleEdit](https://www.nikse.dk/subtitleedit): `subtitleedit` ([SubtitleEdit/subtitleedit](https://github.com/SubtitleEdit/subtitleedit)) - a free (open source) editor for video subtitles - a subtitle editor :&rpar;
    - The latest beta version is also available with `subtitleedit@beta`.

## How do I install these formulae (and casks)?

`brew install mase3206/tap/<formula>`

Or `brew tap mase3206/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mase3206/tap"
brew "<formula>"
```
